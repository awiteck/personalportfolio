All Projects

  • Idea History Agent: AI agent that constructs a DAG model of how different ideas have evolved throughout time and locations.
  • Princeton Senior Thesis: "Rethinking the Hype of Transformer Models: Examples from Electric Load Forecasting": Developed a transformer-based model from scratch that can forecast electricity loads based on historical data and forecasted weather patterns.
  • Radiation: A Deterministic Generative Art Project: Generative art project using p5.js, inspired off the album cover for Flying Lotus's Cosmogramma.
  • Pre-Calculus Textbook: Wrote 160 page advanced precalculus textbook, self-published. Book covers advanced math concepts in support of high school course. Completed and released summer 2019
  • Dlamos: iOS app that uses Spotify API, creating a social media app where users can share music and artists can interact directly with their fanbase, adding a large-scale collaborative element to the music creation process. Over 8k lines of code, fully built-out backend (Spotify authentication, database, etc.) and frontend (SwiftUI).
  • VitalMap: Lead team to build a community-driven Progressive Web Application that eliminates the barriers between students and getting the essential products they need. Completed for Princeton's COS333 class.
  • Greg: A Princeton Course Chatbot: This project uses the Retrieval Augmented Generation (RAG) framework in conjunction with the OpenAI GPT API to create a chatbot, Greg, that can make class suggestions to students based on their interests and preferences.
  • LSH Music Recommendation Engine: In this project, I implemented the locality-sensitive hashing (LSH) vector search algorithm from scratch and used it to efficiently recommend music to a user based on a song they like.
